## Description

This library provides a list of callbacks for Android native lifecycle callbacks.

The callbacks provided are
-- onCreate
-- onDestroy
-- onPause
-- onRestart
-- onResume
-- onStart
-- onStop
-- onLowMemory
-- onTrimMemory

## Usage:

Simply retrieve the singleton ("instance()") and register on the signals provided.
